Páginas de teste (HTML) – Activity Provider (Padrões Estruturais)

Estas páginas permitem testar endpoints sem Postman, no mesmo estilo do repositório anterior.

Configuração

  • GET /config (alias do contrato: /config_url)
  • GET /params (alias do contrato: /json_params_url)

Deploy / user_url

  • Página de teste (GET /deploy?activityID=...)

Analytics

  • GET /analytics/available (alias do contrato: /analytics_list_url)
  • Página de teste (POST /analytics)

Swagger

  • /docs
